# شرح تغییرات در فایل نسخه 2

- بهبود سیستم ثبت‌نام: تفکیک اعتبارسنجی برای ایمیل و نام کاربری جهت نمایش پیام‌های خطای دقیق‌تر (جلوگیری از ثبت تکراری).
- بهبود سیستم ورود: نمایش خطای مشخص برای "کاربر یافت نشد" (در صورت نبود ایمیل) و "رمز عبور اشتباه است" (در صورت عدم تطابق رمز).
- ذخیره‌سازی اطلاعات کاربر جدید در دیتابیس پس از اعتبارسنجی موفق.
